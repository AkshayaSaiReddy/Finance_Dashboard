💰 Finance Dashboard (React)

A modern Finance Dashboard built using React.js that helps users track income, expenses, and visualize financial data in real time.

🚀 Live Demo
finance-dashboard-two-mocha-96.vercel.app

 📌 Features
- Add, edit, and delete transactions
- Real-time balance calculation
- Expense visualization using charts
- Date and time tracking for transactions
- Filter transactions by date
  
 🛠️ Tech Stack
React.js, JavaScript, HTML, CSS, Chart.js

 📂 Run Locally
git clone https://github.com/AkshayaSaiReddy/Finance_Dashboard 
cd finance-dashboard  
npm install  
npm start
